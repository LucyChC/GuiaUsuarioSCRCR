# Roles de Usuario

El sistema SCRCR utiliza roles de usuario para controlar el acceso a los diferentes módulos y funciones.

## Introducción

Cada usuario recibe un rol que determina qué opciones aparecen en el menú lateral y qué acciones puede realizar.

## Objetivo

Asegurar que cada persona use el sistema dentro de su ámbito de responsabilidad, protegiendo la información y agilizando los procesos.

## Roles y descripción

| Rol | Descripción |
| --- | --- |
| Administrador | Supervisa la configuración del sistema y gestiona accesos. |
| Junta Directiva | Consulta reportes y acuerdos estratégicos de la organización. |
| Pastor General | Administra la información espiritual, actas y eventos. |
| Asistente Administrativo | Apoya la operación diaria y la atención de usuarios.

## Funcionalidades visibles

1. Consultar el listado de módulos disponibles.
2. Revisar las tareas asignadas según el rol.
3. Confirmar los permisos antes de ejecutar acciones sensibles.

## Nota sobre visibilidad

!!! note
    Las funcionalidades visibles y accesibles dependen del rol asignado. Algunos módulos o acciones pueden no aparecer si no están autorizados.
