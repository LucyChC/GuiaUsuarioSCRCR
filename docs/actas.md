# Actas

El módulo de **Actas** permite gestionar las minutas y registros oficiales de reuniones y asambleas. Es un espacio para archivar la documentación de decisiones y acuerdos.

## Funciones principales

1. Crear nuevas actas.
2. Consultar actas anteriores.
3. Registrar participantes y acuerdos.
4. Adjuntar observaciones relevantes.

## Procedimiento de uso

1. Ingresa a **Actas** desde el menú principal.
2. Selecciona **Nueva acta** y completa los campos solicitados.
3. Incluye los puntos tratados, acuerdos y responsables.
4. Guarda la acta para que quede registrada en el sistema.

## Consejos de elaboración

- Describe claramente los puntos tratados.
- Indica las decisiones adoptadas y las fechas de cumplimiento.
- Registra a los participantes y los cargos representados.

## Buenas prácticas

1. Archiva las actas vinculadas a cada reunión.
2. Consulta las actas anteriores para dar continuidad a los acuerdos.
3. Mantén un índice de actas por fecha y tema.
