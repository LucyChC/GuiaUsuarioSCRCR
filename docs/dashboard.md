# Menú Principal

Esta es la pantalla principal del SCRCR. Desde aquí el usuario puede acceder a los diferentes módulos del sistema según los permisos asignados.

## Funciones disponibles

- Acceso a los módulos del sistema.
- Navegación rápida entre secciones.
- Visualización de opciones según el rol del usuario.
- Cierre seguro de sesión.
