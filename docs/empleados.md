# Empleados

El módulo de **Empleados** centraliza la información del personal y facilita la administración de recursos humanos. Está orientado a mantener un registro actualizado de los colaboradores.

## Funciones principales

1. Registrar nuevos empleados.
2. Consultar perfiles del personal.
3. Actualizar datos laborales y de contacto.
4. Clasificar según área y cargo.

## Uso habitual

1. Ingresa a **Empleados** desde el menú principal.
2. Busca al empleado por nombre o identificación.
3. Abre la ficha del empleado para revisar sus datos.
4. Realiza los cambios necesarios y guarda la información.

## Puntos clave

- Mantén los datos del cargo y la unidad organizativa actualizados.
- Verifica los datos de contacto y modalidades de jornada.
- Utiliza la información para coordinar tareas y eventos.

## Buenas prácticas

1. Registra nuevas altas lo antes posible.
2. Actualiza la información cuando se produzcan cambios de cargo.
3. Revisa periódicamente el listado de personal activo.
